GET /api/reviews/

Query parametres

The following parameters should be included in the URL path. Request returns all reviews of the movie identified by movie_id. Request returns all reviews from profile identified by profile_id.

Parameter Description
movie_id A unique integer value identifying movie.
profile_id A unique integer value identifying profile.

Example: GET imdb-api.tk/api/reviews/?movie_id=3

Example: GET imdb-api.tk/api/reviews/?profile_id=6

Example: GET imdb-api.tk/api/reviews/?profile_id=6&movie_id=4

Returns review from profile with id profile_id of movie with id movie_id.
