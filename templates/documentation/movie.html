GET /api/movies/

Query parametres

The following parameters should be included in the URL path. Request returns movies with title/genre title that starts with given string. Search is case sensitive.

Parameter Description
title__startswith
genre__title__startswith

Example: GET imdb-api.tk/api/movies/?genre__title__startswith=Horror

Example: GET imdb-api.tk/api/movies/?genre__title__startswith=Action&title__startswith=A

Enables filtering movies by title within one genre.
